The effect system of Eff

How to enable the effect system

To enable the effect system, start Eff with the flag --effects.

The output does not match the expected one

Unfortunately, due to time constraints, the implementation of the algorithm does not yet match the one presented in the paper Inferring Algebraic Effects. The main differences are:

  • The implementation does not pretty-print region parameters, and instead prints region constraints.
  • The implementation does not support instance constants. A workaround is to create appropriate instances with new, though such instances are then labelled by #0, #1, …
  • Occur check is not implemented, so any cyclic constraints result in divergence of the algorithm.
  • Handler types are printed in the extended form even when a compact one exists.
We hope to fix the issues as soon as possible.
